My Work

Autonomous Pick & Place Robot

Technical Lead

We designed parts of a robotic arm in Solidworks. Manufacturing was done via laser cutting acrylic sheet. We performed the assembly of parts with Arduino nano, 4 Servos and Bluetooth module HC-05. I also wrote the code in Arduino IDE for autonomous pick and place of things. We also integrated it with joystick instead of bluetooth.

Staircase climbing robot

Technical Lead

In this project, we designed an acrylic chassis in SOLIDWORKS and wheelbase for a 6-wheel car which looks & works similar to Curiosity Mars Rover. The robot is based on a rocker-bogie mechanism which allows it to climb over obstacles. Made by using 6 geared DC motor, Adafruit Motor driver shield, Arduino Uno and a bluetooth HC-05 module so it can be driven using a smartphone

Obstacle avoiding Path follower Robot

Technical Lead

We designed a robot chassis in SOLIDWORKS. Then we manufactured it by laser cutting it on an acrylic sheet. Then performed the assembly of DC gear motors, L298N motor driver ( Or Adafruit motor driver shield), wheels, IR Sensors, Ultrasonic sensors, Arduino Uno and a bluetooth module HC-05 so it can be driven using a smartphone. Then we executed the code in Arduino IDE. We created both 2-wheel and 4-wheel bots.

Naza-M lite Drone with F450 frame and GPS

CAD Designer

Created a drone with Naza-m lite and F450 frame. Programmed and calibrated it with fly sky i60 remote. Performed calibration and calculations like thrust to weight ratio to achieve maximum stability. Configured Brushless DC Motor, Electronic Speed Controller and pwm.

Self-Balancing Robot

Technical Lead

This is a heavy programming project, which requires integration of PID Control and the use of Kamlan filter for smoothing the transitions of sudden movements. Made with two motors, L298N motor driver (Or Adafruit motor driver shield), an Arduino Uno, MPU-6050 Gyroscope sensor and Accelerometer.

Semi-Automatic Plastering Machine

Technical Lead

In this project, we used a pneumatic cylinder as the drive-train to lift a plaster-head which applies the plaster onto the wall. Furthermore, we integrated an Arduino with a Bluetooth chip HC-05, to control the plastering process using a smart phone.
